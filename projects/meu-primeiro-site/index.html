Quem é Lucas Back?

Sou estudante de Análise e Desenvolvimento de Sistemas na Cruzeiro do Sul, no último semestre do curso. Meu foco principal é o desenvolvimento front-end — criar interfaces que combinam bom design com código bem escrito.

Este foi meu primeiro projeto criado para praticar HTML, CSS e JavaScript do zero. Cada linha aqui foi escrita manualmente para entender como as tecnologias web funcionam juntas.

Estou buscando minha primeira oportunidade júnior em front-end, onde possa crescer dentro de uma equipe e contribuir com o que já sei enquanto aprendo com profissionais mais experientes.
